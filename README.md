# 💧 Liquid Checker

**Liquid Checker** — это утилита для глубокой проверки целостности системы. Проект объединяет надежные алгоритмы сканирования с современным асинхронным интерфейсом, выполненным в уникальном стиле «Liquid Glass». 

Разработано для быстрого выявления аномалий, модифицированных системных файлов и потенциальных угроз без перегрузки ресурсов ПК.

## ✨ Ключевые особенности

* **Проверка целостности системы:** Быстрое сканирование критически важных директорий и сравнение хеш-сумм файлов.
* **Liquid Glass UI:** Эксклюзивный пользовательский интерфейс с эффектами полупрозрачности и плавными анимациями. Никаких скучных серых окон.
* **Асинхронный движок:** Интерфейс не зависает во время тяжелых проверок благодаря грамотному разделению потоков.
* **Легковесность:** Минимальное потребление оперативной памяти при работе в фоновом режиме.

## 🛠 Технологический стек

* **Язык программирования:** Python 3.10+
* **GUI Фреймворк:** [Flet](https://flet.dev/) (построение современного реактивного интерфейса)
* **Архитектура:** Асинхронный I/O
